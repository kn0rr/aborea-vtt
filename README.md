# Aborea Lite für Foundry VTT

Ein kleines System-Template für **Foundry VTT v13** mit:

- Actor-Typen: `character`, `npc`
- Item-Typen: `talent`, `weapon`, `gear`
- einfacher Charakterbogen
- explodierendem W10 für Attributs- und Talentwürfe
- `template.json` als Grundlage für weitere Systemdaten

## Installation lokal

1. Ordner `aborea-lite` nach `Data/systems/` kopieren.
2. Foundry neu starten.
3. Beim Erstellen einer Welt `Aborea Lite` als System wählen.

## Nächste sinnvolle Ausbaustufen

- eigene Item-Sheets
- Schadenswürfe für Waffen
- Initiativelogik / Combat Tracker Integration
- Lokalisierung erweitern
- Klassen- und Magieregeln aus Aborea ergänzen
- Icons und Compendien hinzufügen

## Wichtiger Hinweis

Das ist **kein vollständiges offizielles Aborea-System**, sondern ein bewusst kleines Startgerüst zum Weiterbauen.
